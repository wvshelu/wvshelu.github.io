Programs

Below are some of the programs I worked on, either individually or in a team.

Muses

The website, Muses, allows users to upload advertisements or their artwork and sell them on the website. I worked with two other people to create this website for the FBLA E-Business event. I created the functionalities, such as the uploading function, the account system, and the search function.

View Muses

Éclat

The mobile application, Éclat, allows users to share images of their suits for an FBLA event so others can give them feedback. I worked with two other people to create this app for the FBLA Mobile Application Development competition. I helped connect the application template to the database.

Fire and Water

The Java project, Fire and Water, is a traditional shooter game in which the user must kill off the "Fire Balls" before they kill the user's "Water Ball." I created this project for my final project in Introduction to Programming.

Homestead CSF Website

The HHS CSF Website helps broadcast volunteer events and regulate the members' hours. AS the CSF Webmaster, I added multiple features, such as the waitlist feature, the officer tab, and the online application, and maintained the website by fixing the about officers page, correcting the members' hours, etc.
